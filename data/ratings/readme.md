# Subjective Ratings

This directory contains anonymized subjective rating tables.

- `subjective_ratings_raw.csv`  
  Raw subjective ratings collected after each taste stimulus.

Each row corresponds to one `(subject_id, taste_type)` pair and can be aligned with:
- `data/videos/<subject_id>/<taste_type>.mp4`
- `data/frames/<subject_id>/<taste_type>/`

All personal identifiers have been removed. Participants are indexed as S001, S002, ...
